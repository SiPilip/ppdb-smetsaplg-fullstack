Saya ada sebuah project sistem penerimaan peserta didik baru sma methodist 1 palembang, sebelumnya seluruh sitem dilakukan melalui offline, tetapi klien meminta agar semua sistem dapat dilakukan secara daring, saya ingin anda membantu saya untuk melakukan planning pasca program, baik dalam perancangan databse, struktur folder front-end (next.js fullstack) dan backend (mongo.db integrated with next.js), Saya ingin kita merancangkan apakah sistem perancangan menggunakan sistem midtrans (otomatis) dengan resiko, atau dengan menggunakan cara manual (pay, dan give the invoice) melalui whatsapp, saya ingin kita juga mebangun sebuah sistem yang dapat terintegrasi dengan whatsaap api, jadi seluruh informasi (update pendaftaran) dapat melalui api whatsapp. Seluruh pendaftar (user) harus mendaftar terlebih dahuli menggunakan email baru dapat melakukan proses registrasi, aplikasi terdapat dashboard yang menampilkan informasi bagi user, terdapat otp melalui nomor telepon (via wahtsapp).
